Facebook Instant Article - SF Fed Blog

Custom Instant Article styles for SF Fed Blog. Refer to Facebook Instant Articles documentation for more information.

  1. Header image

    Place <img> tag inside <header>.

  2. Logo

    Use logotype.

  3. Kicker

    Normally headcrumb on webpage, kicker indicates section or user position within hierarchy. Use <h3 class="op.kicker">, which should be inside the <header> section of the body. Style with <color="#883529">.

  4. Title

    Within the <header>, style with <h1 color="#333333">.

  5. Post date

    Date must be tagged with <address> within a <header> tag.

  6. Body

    Use backup font, Helvetica Neue, for Instant Article principal font, <color="#333333">

  7. Footer

    Within the <footer> tag, use <aside> for main footer info and <small> for copyright text. (Default social sub-footer automatically appended.)
